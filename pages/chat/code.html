不可思議！廚房裡的這些東西真能美白除斑？皮膚科專家：這個方法效果最好最省錢

日期：2020-02-21 作者：林 慧

最新調查發現，有70%的女性認為祛斑產品沒用，而只有30%的女性認為祛斑產品有用。除了祛斑產品，祛斑效果較好的天然除斑方法有哪些？那么到底什么樣的祛斑方法才是最有效的呢？ 小編帶大家一起來看下吧！

TOP6 - 絲瓜祛斑法

絲瓜中含有豐富的維生素和蛋白質，能夠幫助肌膚在凈白的同時去除斑點，同時，絲瓜中的微量元素以及抗壞血酸，對肌膚都起著非常好的保健作用，可以說絲瓜祛斑的效果是很棒的。

將絲瓜曬乾後，磨成粉末，然後每晚用溫水，再加入適量的蜂蜜，進行混合，調成糊狀，塗抹在麵部肌膚上，30分鐘後清洗乾凈，然後美美的睡上一覺，絲瓜加入蜂蜜，可以令肌膚在祛斑的同時更加潤滑，有助於去除皺紋，令肌膚細嫩白滑，堅持使用4-5個月，可以淡化黑斑。

TOP5 - 茄子祛斑法

茄子是我們日常生活中非常常見的蔬菜之一，它富含維生素P，可以幫助軟化血管，同時還富含維生素C，可以幫助肌膚預防黑色素的形成，對肌膚有著很好的美白功效。

將茄子清洗乾凈，切成小塊，放入榨汁機內，加入適量的清水，榨成汁後塗抹在麵部肌膚有斑點的地方，每日早晚各一次，大約2個月左右的時間斑點就會慢慢自行消退了，但茄子中具有酚氧化酶易氧化形成黑色素，容易適得其反，不推薦使用。

TOP4 - 白醋祛斑法

相信很多想要除斑的人都聽說過白醋除斑法，適量濃度的白醋塗抹在色斑生長的地方，每天兩次，大約1個月就能見到效果

白醋的主要成分是醋酸，它是一種酸性物質，利用其具有腐蝕灼燒性，從而達到祛除斑點的目的，但長時間使用，會出現瘙癢的癥狀，甚至會破壞皮膚的障礙功能，形成敏感肌，務必謹慎使用！

TOP3 - 冬瓜祛斑法

此方法在古書文獻中早有記載，古人擅長用冬瓜減肥美容，將冬瓜仁曬乾，研細末，每晚臨睡時，取適量與水調和，塗洗麵部，堅持約2個月，有消除色斑，潤澤皮膚作用

由於該方法需要大量冬瓜仁，鮮有人去嘗試，具體效果如何？是否有副作用？暫時無從得知，不建議屏幕前的姊妹拿自己的臉做實驗哦

TOP2 - 檸檬祛斑法

美眉們都知道，檸檬中含有豐富的維生素C，以及多種微量元素，對於呵護肌膚來講，這些都是非常好的美容養顏的物質，檸檬可以幫助嫩白肌膚、淡化色斑，效果神奇。

檸檬雖然有一定的淡斑作用，但是爲感光性食物，不宜在白天食用，檸檬還有強烈的果酸，食用容易傷到腸胃。醫師建議檸檬不要敷臉，皮膚易過敏，且對角質層會造成傷害。

小編一直以爲「天然除斑法」是最安全的除斑方法，但查閱相關文獻和諮詢醫師後才恍然大悟，原來「天然除斑法」並非安全！！

到底怎樣除斑最好？

1.雷射除斑： 見效快，費用昂貴、還太傷膚、易反黑、風險高，明星都不做雷射除斑；

2.天然除斑： 省錢，安全性無保障、步驟繁瑣、時間長、見效慢，歲月不等人，女人如花的年紀就那麼幾年；

3.祛斑產品： 不可否認，有一些祛斑產品的確安全有效，但大部分爲了營銷，誇大除斑效果，還會過量添加化學成分，使用結果可想而知！
